## 🚀 Funcionalidades

- **Modo PvP Local** - Jogue contra um amigo no mesmo dispositivo
- **Modo Contra CPU** - Desafie a IA com movimentos inteligente
- **Tabuleiro Visual** - Interface elegante com design "glassmorphism"
- **Sistema de Sons** - Efeitos sonoros para movimentos, capturas, xeque e fim de jogo
- **Histórico de Movimentos** - Registro visual das jogadas
- **Destaque do Último Movimento** - Visualização clara da última jogada

## 📦 Instalação

```bash
# Clone o repositório
git clone https://github.com/sirmarciusx/-master-xadrez-pro.git

# Entre na pasta
cd master-xadrez-pro

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```

## 🎨 Design

- **Tema**: Dark mode com accents em dourado (#f2ca50)
- **Estilo**: Glassmorphism com efekto de metal escovado nas casas
- **Fontes**: Playfair Display (display) + Inter (corpo)

---

## 🤖 IA da CPU

A CPU utiliza um algoritmo semi-inteligente que:
- Prioriza capturas de peças de maior valor
- Considera jogadas en passant
- Protege o rei quando está em xeque
